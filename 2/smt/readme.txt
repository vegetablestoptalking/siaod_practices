Установка Z3
============
1. Получить путь к Питону. В консоли IDLE набрать: import sys; print(sys.path[-1])
2. В консоли cmd.exe перейти в основную папку Питона. Оттуда -- в /Scripts.
3. pip install z3-solver
Альтернативный вариант:
    3. pip install pysmt
    4. pysmt-install.exe --z3
5. Выйти из IDLE!

Шпаргалка по Z3
===============
- Int("name"). Создание целочисленной переменной с внутренним именем name.
- IntVector("name", n). Создание n Int-переменных. Имена генерируются с автоматическим добавлением индексов.
- s = Solver(). Создание объекта SMT-решателя.
- s.check(). Проверка выполнимости. Выдается sat, unsat или unknown.
- s.model(). Если получен sat, то будет выдана модель -- состояние переменных.
- s.add(...). Добавить правило, выполнимость которого будет проверяться.
- Sum([a, b, c, ...]). Сокращение для записи вида a + b + c + ....
- Distinct([a, b, c, ...]). Правило, означающее, что переменные a, b, c, ... должны иметь различные значения.
- s = Optimize(). Используется вместо Solver() для задач оптимизации.
  В этом случае добавляется s.maximize(...) или s.minimize(...) с указанием переменной или выражения, подлежащего оптимизации.

В правилах могут использоваться обычные арифметические операции и операции сравнения.
Булевые операции задаются в виде: And(), Or(), Not().

Задачи с использованием Z3
==========================
1. На лужайке 9 животных, кролики и фазаны. Видны 24 лапы. Сколько на лужайке кроликов и сколько фазанов?
2. Среди последовательности целых nums найти подмножество длины 3, сумма значений которого равна 0.
3. Найти магический квадрат 3x3, сумма соотв. значений которого равна 42.
4. Задача о рюкзаке 0-1.
5. Раскрасить карту (см. map.png).
